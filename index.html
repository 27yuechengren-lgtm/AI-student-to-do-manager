Problem and Solution

❌ The Problem

Many students struggle to manage homework, revision, projects, and daily tasks. Existing tools often:

  • Are overly complex
  • Require paid subscriptions
  • Lack intelligent features

Result: Wasted time, missed deadlines, increased academic stress

✅ Our Solution

A simple, free AI-powered task manager featuring:

  • Automatically categorize tasks as homework, exams, personal, etc.
  • Eliminate manual input
  • Keep everything organized

Result: Save time, reduce stress, focus on what matters

Feature Demonstration

📝 Basic Task Operations

  1. Enter a Task: Type your task in the input field (e.g., "Complete math homework")
  2. Add to List: Click the "Add Task" button to add it to your list
  3. View Tasks: Your new task appears immediately in the active tasks list
  4. Mark Complete: Click on the task text to mark it as complete
  5. Delete Task: Click the "X" button to remove a task

Development Challenges and Solutions

🧠 AI Categorization Accuracy

❌ Problem: Initial keyword matching was inaccurate
✅ Solution: Expanded keyword dictionary, converted input to lowercase for better matching

💾 Data Persistence

❌ Problem: Tasks were not saving to localStorage correctly and were lost on refresh
✅ Solution: Debugged JSON serialization, ensured save function is called after every state change

📱 Responsive Layout

❌ Problem: Buttons overlapped and layout broke on small screens
✅ Solution: Restructured layout with CSS Flexbox, proper spacing for adaptive behavior
